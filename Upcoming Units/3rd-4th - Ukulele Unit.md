## Day Two: Plucking strings, strumming, and the C chord

| ***Part of Lesson*** | ***Plan for Today***                                                                                                                                                                                                                                                                                                                                                                                                      | ***Resources/Links*** |
| -------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------- |
| Concept              | [[Ukulele]]                                                                                                                                                                                                                                                                                                                                                                                                               |                       |
| Method               | Practice picking individual strings with a repetition game, then get used to strumming a steady beat on open strings. Finally, learn a C chord and practice strumming.                                                                                                                                                                                                                                                    |                       |
| Warm-up              | Everyone gets a ukulele (tune them beforehand!) and practices holding it correctly. Go over the note names of each string one by one, using a repetition game to practice picking notes. For example, I will play a rhythm on the G and instruct them to copy my pattern. I will do this a few times until I move onto the C string, E string, and A string respectively. For the challenge rounds, do G+C, C+E, and E+A. |                       |
| Execution            | Show proper strumming posture and instruct students to copy me. Strum a steady beat on open strings and exaggerate the motions a bit. Add in up beats for a DUDUDUDU pattern. Then show them a C chord chart. Ask if anyone knows how to read chord charts and have them show the class. Remind them of the numbered fingers and order of the strings. Get everyone to play a C and practice some strumming patterns.     |                       |

## Day Three: Adding the Am chord

| ***Part of Lesson*** | ***Plan for Today***                                                                                                                                                                                                                                                                                                                                                        | ***Resources/Links***                       |
| -------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------- |
| Concept              | [[Ukulele]]                                                                                                                                                                                                                                                                                                                                                                 |                                             |
| Method               | Learn the Am chord and add it into a strumming pattern with C.                                                                                                                                                                                                                                                                                                              |                                             |
| Warm-up              | Play the string picking repetition game on the challenge level.                                                                                                                                                                                                                                                                                                             |                                             |
| Execution            | Practice the C chord again and refresh where needed. Now show an Am chart and have students practice strumming on it. Practice switching between the two by slowly counting and having students switch between the chords on the number one. If they do well with it, have them challenge themselves with the video. If they get off track, just pick up wherever they can! | https://www.youtube.com/watch?v=fY8K5_3x-64 |
